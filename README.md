# Arul's Digital Universe

A highly creative, futuristic portfolio website. Built with React, Vite, Tailwind, Three.js, Framer Motion, and GSAP.

## Run

```bash
npm install
npm run dev
```

Then open the URL Vite prints (default http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Stack

- Vite 5 + React 18 (JSX)
- Tailwind CSS 3
- three + @react-three/fiber + @react-three/drei
- framer-motion + gsap (with ScrollTrigger)
- lucide-react + react-icons (Simple Icons)
